Internet Speed Dashboard

A real-time Internet Speed monitoring dashboard built with PHP, Bootstrap 5, and Chart.js. This project measures your internet connection's download speed, upload speed, and ping, and displays them in interactive charts and stat cards. The dashboard also shows the last 10 measurements in a table.

Features

- Max/Min/Avg stats displayed in colorful cards
- Download & Upload Speed chart
- Ping chart
- Last 10 measurements table
- Fully responsive design with Bootstrap
- Easy to run locally using XAMPP

Technologies Used

- PHP – backend for fetching data from speed_test.php
- JavaScript – dynamic data fetching and Chart.js rendering
- Chart.js – interactive line charts for speed and ping
- Bootstrap 5 – responsive UI, cards, tables, and layout
